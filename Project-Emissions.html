Making emission tracking easier

Problem

A big international multi industry corporation realised the need for a proper internal tool track production emissions. As the need arised from the new regultaions they also saw it as a possible business opportunity and decided to have a POC to pitch the tool to the clients to see whether it would be a good portfolio piece for them.

Solution

As a team of three in a tight 6-weeks timeline we created dummy website covering the base functionality of a tool plus a number of mockups showcasing additional possible features. We focused on the ease of use as we pinpoited that the main challenge with the tool would be it's occasional use by different people so everything should be clear for any first-time user.

Results

The tool was not only green lighted internally but also received genuine interest from the clients so there is an ongoing work on a production-level version of the tool.

Learnings

  • Project success was the result of efficient communication within a lean team and a proper preparation phase;
  • Strict deadline with the feature list catered to what's possible within that timeframe helped keep work and discussions with the stakeholders focused and productive;
