Photo editor app

A friend of mine who is Android developer approached me with his library of photo filters and the idea to produce a photo editing application.

Before starting we needed to come up with the strategy and requirements. To help with that we conducted a competitor analysis and ran a small Typeform survey. As a result we were able to narrow down the main points we should be focusing on:

1

Application stability and performance

2

File quality after editing process — we are not cropping images and allowing to save files at original resolution (up to 4096px x 4096px)

3

Clean and user-friendly user interface — many users are confused by too many options, complicated onboarding or apps looking like a pro tool

4

Give the users feeling of being in control — while we decided to keep the app simple and filter-focused we are still adding essential edit options


As for the user interface the main idea was to keep it simple yet let it have some personality. So I've picked a dark theme colour palette with one accent colour assuring it passes the color contrast check for the labels.

Also, I've designed a small set of icons to ensure the iconography is consistent and fit the overall design.

At the moment the application is in Beta with full release planned in late December 2020.
