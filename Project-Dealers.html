CMS for auto dealers

One of the biggest European automotive businesses realised they are missing out on part of the dealer's market due to some missing features. As a result they decided to rewrite their existing CMS to redesign it including those features.

This project was a great example of 'if it works, don't break it' so it was essential to keep all the existing customers by granting feature parity and slightly improved UX while also introducing additional features to expand the client base. To make sure both conditions are met the project timeline included plenty of time for the user interviews and testing. As a sole designer I worked closely with the PO in short sprints to allow us to have features ready to prototype testing as often as possible. Simultaneosly I was working with the developers to ensure the implementation of the approved features goes as smoothly as possible.
